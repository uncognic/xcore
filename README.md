# Xcore
### Overview
A hobbyist operating system for i686 architecture, written from scratch in C and Assembly.
### Features
- x86 i686 Support – Fully functional on 32-bit Intel architectures.
- Custom Kernel – Built from scratch with no external OS dependencies.
- GDT & IDT – Global and Interrupt Descriptor Tables for protected mode operation.
- Interrupt Handling
- VGA Terminal – Basic text-mode output with scrolling support.
- Keyboard Input – PS/2 keyboard driver for user input.
- Paging & Memory Management – Implements basic paging and memory protection.
- Shell (kshell) – Minimal shell to interact directly with the kernel.
- Bootloader Compatibility – Works with standard multiboot bootloaders.
